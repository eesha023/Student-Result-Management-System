# Student Result Management System (C)

This project is a C program that manages student results.

## Features
- Stores student details (roll number, name)
- Accepts marks with validation
- Supports different maximum marks per subject
- Calculates total, percentage, and grade
- Uses structures, union, and typedef

## Language Used
- C

## Concepts Used
- Structures
- Union
- Typedef
- Arrays
- Input validation
- Loops and conditions

## How to Run
1. Compile the program using a C compiler
2. Run the executable
3. Enter student details and marks as prompted
